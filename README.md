# Edviron_assignment
This project is an interactive revenue dashboard built using Excel. The main objective of the dashboard is to analyze transaction data and provide clear insights about revenue, partners, gateways, and user behavior.
The project is structured in three layers:
Raw Data (tbl_transactions) – Contains cleaned transaction data.
Modelling Sheet – Performs all calculations using formulas like COUNTIFS and SUMIFS.
Dashboard Sheet – Displays final results in the form of KPI tiles and charts.
The dashboard includes key performance metrics such as:
Total Transactions
Total GMV (Gross Merchandise Value)
ERP Revenue
Edviron Gross Revenue
Edviron Net Revenue
Pending Exposure
Unique Users
Frequency of payment per user
These metrics help understand overall business performance and revenue flow.
The dashboard also includes visual charts for better analysis:
Revenue split between ERP and Edviron
Time-based revenue trend (daily/monthly)
Gateway-wise contribution
Payment method mix
Interactive slicers are added for:
Date range
ERP (Partner)
Gateway
Payment method
When any filter is changed, all KPI tiles and charts update automatically. No manual filtering is used. The dashboard is fully dynamic and driven by formulas and pivot connections.
This project demonstrates skills in:
Data cleaning and structuring
Revenue calculation logic
Excel formulas (COUNTIFS, SUMIFS)
Pivot tables and charts
Interactive dashboard design

Overall, the dashboard provides a clear, professional, and dynamic view of revenue performance and business insights.
